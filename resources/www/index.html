webserv

An NGINX-inspired HTTP/1.1 server in C++98.

Try the demo routes

  • /upload/ — autoindex; POST a multipart/form-data file here.
  • /cgi-bin/hello.py — Python CGI.
  • /cgi-bin/hello.php — PHP CGI (requires php-cgi).
  • /redirect — 302 redirect back to /.
  • 127.0.0.1:9090/ — second virtual host, autoindex of resources/cgi-bin.

Configuration

This page is served by resources/default.conf. Edit it (or write your own) and restart with ./webserv from the repo root.

See README.md for the full configuration reference.
